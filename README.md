# Football App
Football app about Serie A

## Information
This app is a football league information of Serie A (Italian league)

The information contain in this app
- Standing
- Team Info
- League Matches
- Favorite team

## How to Install
Install node package
```sh
$ npm install
```

## How to Run
Run development server
```sh
$ npm run start
```

Build production
```sh
$ npm run build
```

Build and preview production
```sh
$ npm run preview
```
